# Decks

Generated image-based decks for the DevDays 2026 sessions.

Each deck directory generally contains:

- `deck.md`: talk arc and slide plan.
- `visual-brief.md`: visual system and generation guardrails.
- `prompts/`: deck-level and slide-level image prompts.
- `slides/`: canonical raw slide images named `slide-NNN.png`.
- `preview.png`: contact-sheet style preview.
- `deck.html`: self-contained browser deck.
- `deck.pptx`: PowerPoint export.
