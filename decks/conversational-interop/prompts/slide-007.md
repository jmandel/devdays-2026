# Slide 7 Prompt

Title: `Three Layers, Three Jobs`

Do not draw this as one vertical stack shared by everyone. The point is two different organizational agents talking to each other, while each agent remains grounded in its own tools and its own system of record.

Main layout:

- Top / center: `A2A task dialogue` as a bidirectional conversation between two distinct agents.
- Left agent card: `provider liaison agent`.
- Right agent card: `payer policy agent`.
- Between the agents: bidirectional message arrows with small protocol chips: `Agent Card`, `message/send`, `tasks/get`.
- Under the provider agent: a tool row labeled `trusted tools` with chips `search notes`, `query FHIR`, `cite evidence`.
- Under those provider tools: a source row labeled `system of record` with `EHR`, `FHIR`, `clinical notes`.
- Under the payer agent: a tool row labeled `trusted tools` with chips `policy lookup`, `criteria check`, `status update`.
- Under those payer tools: a source row labeled `system of record` with `coverage policy`, `claims/admin`, `authorization queue`.

Bottom callout exact text: `A2A connects agents. Tools connect each agent to its own authority.`

Keep the existing deck style: white grid canvas, bold black title in the same position, thin blue/teal/green/purple linework, rounded white cards, no logos, no robots, no generic AI brain imagery. Make the architecture read left-to-right across organizations, not top-to-bottom as a shared technology stack.
