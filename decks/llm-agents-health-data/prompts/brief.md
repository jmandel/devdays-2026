# Prompt Brief: LLM Agents With Health Data

Create a 16:9 image-based workshop deck in a consistent technical workbench style.

Use the approved slide 1 as the typography and layout reference for the rest of the deck: broad heavy non-condensed sans-serif title, top-left title placement, white subtle grid canvas, clean technical cards, thin colored connector lines, readable monospace chips, and the bottom progress rail.

Hard typography rule: never use condensed, squeezed, horizontally compressed, or narrow-kerning title text. For long ideas, split into short title plus subtitle.

Palette: white background, charcoal text, muted gray linework, SMART/FHIR blue `#2B78C5`, trust green `#48A868`, workflow amber `#F2A23A`, agent purple `#7A3E91`, caution red `#D84747`, terminal charcoal `#24292F`.

Recurring rail: `connect`, `request`, `analyze`, `build`.

Avoid: recurring logos, fake EHR/hospital branding, robot/brain AI clip art, magic arrows from data to insight, generic summaries, fake patient facts, clinical advice claims, and tiny unreadable paragraphs.
