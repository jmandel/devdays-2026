## Cross-Talk Patterns

- Slides should mostly explain mental models: launch vs. authorization, artifact vs. flow, patient app vs. provider system, tool-use vs. agent-to-agent dialogue.
- Live demos should carry the proof. Each prep file includes fallback screenshots/static artifacts because several demos depend on current browser, wallet, network, or sandbox behavior.
- The strongest continuity across the week is: SMART gets data moving, patient-shared artifacts reduce repeated intake, digital credentials improve presentation UX and privacy, skills/tools let agents work on real records, and conversational interop handles the long tail that fixed schemas do not.
- For current/future milestones, be careful with dates. The prep files flag draft/WIP items and future dates, especially CMS framework milestones and draft standards.
